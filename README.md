**Met** provides several macros extension traits that make working with [`proc-macro2`] easier. 

# Setup

Add this to your `Cargo.toml`:

```toml
[dependencies]
met = "0.1.0"
proc-macro2 = "1"
```

[`proc-macro2`]: https://crates.io/crates/proc-macro2
